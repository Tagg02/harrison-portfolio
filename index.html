Python TCP Port Scanner

A Python-based TCP port scanner that identifies open ports on a target host, demonstrating network enumeration techniques and basic attack surface awareness.

  • Language: Python
  • Core concepts: TCP/IP, sockets, port scanning
  • Functionality: Host input, configurable port ranges, open-port reporting
  • Security relevance: Identifying exposed services during reconnaissance
  • Planned improvements: Multithreading, timeouts, banner grabbing

⚠️ Scanning should only be performed on systems you own or have explicit permission to test.

HTML Carbon Footprint Calculator

A client-side web application that estimates an individual’s annual carbon footprint (tCO₂e/year) using common lifestyle inputs, with a focus on clean UI design and privacy-first implementation.

  • Technologies: HTML, CSS, JavaScript (no frameworks)
  • Inputs: Driving distance, electricity usage, flights, diet
  • Logic: Simplified emissions factors combined into an annual estimate
  • UX focus: Clear layout, inline guidance, instant feedback
  • Privacy-first: All calculations run locally in the browser
  • No backend: No accounts, tracking, or data storage
