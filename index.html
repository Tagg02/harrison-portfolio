Python TCP Port Scanner

A Python-based TCP port scanner that identifies open ports on a target host. This project demonstrates core networking knowledge (TCP/IP + sockets) and how exposed services can form an attack surface.

  • Technologies: HTML, CSS, JavaScript (no frameworks)
  • Inputs: Driving distance, electricity usage, flights, diet category
  • Logic: Simplified emissions factors combined into an annual estimate
  • UX focus: Clear layout, inline guidance, instant feedback
  • Privacy-first design: All calculations run locally in the browser
  • No backend: No accounts, no tracking, no data storage
  • Limitations: Uses approximate emission factors for demonstration purposes
